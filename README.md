# EyeBox Smart Display (ESP32)

Анимированный дисплей с "глазами" на базе ESP32, реагирующий на наклон, движение и бездействие.

Проект использует TFT-дисплей ST7789 (240x240), гироскоп/акселерометр (MPU6050), модуль часов DS1307 и WiFi-синхронизацию времени.

---

## 🚀 Возможности

- Анимированные глаза с эмоциями  
- Реакция на наклон устройства  
- Отображение времени и температуры  
- Синхронизация времени через WiFi  
- Idle-анимации и режим сна  

---

## 🧩 Компоненты

- ESP32 (рекомендуется S3)
- TFT дисплей 240x240 (ST7789)
- MPU6050
- DS1307 (RTC)
- Подсветка дисплея (GPIO)

---

## 📁 Структура проекта

- `ESP32S3-Axel.ino` — основной скетч  
- `Eyes.h` — логика глаз и анимаций  
- `User_Setup.h` — настройки TFT_eSPI под дисплей ST7789 (240x240)

---

## ⚙️ Быстрый старт

1. Установите библиотеку **TFT_eSPI**  
2. Замените файл `User_Setup.h` на версию из репозитория  
3. Укажите WiFi данные в скетче  
4. Загрузите код на ESP32
